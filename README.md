# FFmpeg-Converter-GUI
FFmpeg Converter GUI v2.6 (V2.6 is mainly for RTX GPUs)
FFmpeg Converter GUI v3.0. 
v3.0 adds hardware encoder support for AMD AMF and Intel Quick Sync alongside the existing NVIDIA NVENC,
with a per-GPU capability filter that only offers codecs your specific GPU can actually encode in hardware
(note: v3.0 has not yet been as extensively tested as v2.6).
